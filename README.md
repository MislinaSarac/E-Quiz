# E-Quiz


## E-Quiz Uygulaması

Uygulamanın amacı: Sınav sorumlusunun soru ekleyip, Admin'in de yönetebildiği ve   Öğrencinin de soruları çözerek kendinini test ettiği bir sistem. Kullanıcı bilgileri ve sorular SQL' de saklanmaktadır. Uygulama OOP kullanılarak Visual Studio ve SQL Server Management Studio kullanılarak C# dilinde geliştirilmiştir.
